== Description ==

This plugin allows you to enter reviews about your company as a custom post.
Only the user empowerment can add reviews.
The post should contain the client's name as Title, photo as Post thumbnail and review text as Body message.
The result is shown as a jcarousel in widget.
The widget has the following settings: title, scheme, background. 

== Installation ==

Upload the Review Easy plugin to your blog, activate it and add some reviews. 
Place the widget anywhere on the page
